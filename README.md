# ProyectoAPI
Este proyecto de Data Science explora el canal oficial de YouTube de Billie Eilish mediante la API de YouTube Data v3. El objetivo principal es extraer insights relevantes sobre su actividad, popularidad y evolución en el tiempo a partir de sus videos publicados.

🎯 Objetivo
Analizar el comportamiento de publicación, interacciones y engagement del canal de Billie Eilish, con el fin de responder preguntas clave como:
¿Cuáles son los videos más populares del canal?
¿Cómo ha evolucionado su frecuencia de publicación?
¿En qué años o meses ha publicado más contenido?
¿Qué videos generan mayor interacción?
¿Existen picos de actividad cerca de eventos o lanzamientos clave?

🧰 Herramientas utilizadas
Python (Google Colab)
YouTube Data API v3
Pandas, NumPy, Seaborn, Matplotlib
WordCloud, datetime
Google Developers Console (API Key)

🧪 Flujo del proyecto
Extracción de datos desde la API de YouTube.
Limpieza y transformación del dataset.
Ingeniería de atributos para enriquecer las variables.
Visualizaciones exploratorias para responder preguntas de interés.
Insights clave basados en datos reales.

📈 Ejemplos de visualizaciones
Evolución mensual de publicaciones
Publicaciones por año y trimestre
Top 5 videos por vistas, likes y comentarios
Comparación de actividad vs. eventos clave
